# jenktrace v1.0.9

jenktrace 1.0.9 replaces the build list card layout with a shadcn Table so build status, build number, and duration render in fixed vertical columns across recent builds.

## Highlights

- Replaced the build list card stack with a shadcn Table in the job details view.
- Aligned build status, build number, and duration into dedicated table columns.
- Kept per-build log access available from the new table row actions column.

## Included in this release

- Added the shadcn Table UI component to the main view component library.
- Updated the Jenkins details panel build list rendering to use table rows instead of free-form flex cards.
- Bumped packaged app version metadata to 1.0.9.

## Notes

- Build filtering still applies only to the build list and does not change analytics cards or charts.
- The build name column still shows the started timestamp below the build title for each row.
