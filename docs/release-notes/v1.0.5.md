# jenktrace v1.0.5

jenktrace 1.0.5 makes Jenkins instances easier to distinguish by letting operators set a custom name, icon label, and icon background color for each saved instance.

## Highlights

- Added optional per-instance sidebar identity settings for a custom display name, short icon label, and icon background color.
- Added a live identity preview inside the instance dialog so operators can see how each instance will appear before saving.
- Updated the instance sidebar and instance detail card to show the custom identity while still preserving the Jenkins hostname for context.

## Included in this release

- Persisted the new identity fields in the shared instance model, local config storage, and save flows that also update jobs.
- Normalized icon labels and hex colors before saving, with validation for invalid color input.
- Updated the main details header to show the custom instance display name when no job is selected.
- Bumped packaged app version metadata to 1.0.5.

## Notes

- Custom icon labels are normalized to uppercase alphanumeric text and limited to three characters.
- Leaving the new identity fields blank preserves the existing hostname-based behavior.
