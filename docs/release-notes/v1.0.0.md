# jenktrace v1.0.0

jenktrace 1.0.0 is the first stable release of the desktop app for monitoring Jenkins jobs from a focused local workspace.

## Highlights

- Monitor multiple Jenkins instances from one desktop app.
- Add, edit, test, and remove Jenkins connections without leaving the app.
- Track specific jobs per instance, including nested Jenkins folder paths.
- Review job details, recent activity, and build analytics for selectable time ranges.
- Inspect build health with success, failure, and running status summaries.
- Prefetch and retain build logs for configured job states.
- Run periodic background polling for monitored jobs with configurable intervals.

## Included in this release

- Updated app identity and packaging under the `jenktrace` name.
- App icon and window title updates across the desktop app and packaged artifacts.
- GitHub Actions release flow for tag-based builds and attached release artifacts.

## Notes

- macOS local release builds currently publish packaged artifacts without a DMG in this repository configuration.
- Release artifacts are attached automatically when a GitHub tag is pushed.
