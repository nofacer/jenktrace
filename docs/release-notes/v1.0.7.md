# jenktrace v1.0.7

jenktrace 1.0.7 hides the app log panel by default and moves log access into a bottom status bar toggle so the main job details view stays focused until logs are needed.

## Highlights

- Hid the app log panel on first load so build analytics and build history remain the primary view.
- Added a bottom status bar toggle with a log icon that opens the app log panel on demand.
- Kept app log refresh controls and recent-entry summaries available from the collapsed status bar and expanded panel.

## Included in this release

- Moved app logs out of the main details card into a collapsible bottom drawer in the details panel.
- Added a persistent status bar row that shows app log availability and toggles the drawer open or closed.
- Bumped packaged app version metadata to 1.0.7.

## Notes

- App logs remain auto-refreshed while the window is visible.
- The status bar keeps a recent-entry count visible even while the log panel is collapsed.
